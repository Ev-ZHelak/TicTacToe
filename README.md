## Introduction

TicTacToe is a classic paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner.

This project provides a simple implementation of the game with options to play against a computer opponent.
